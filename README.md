# 学习作业

### 第一周
1. 使用ES6完成点赞+1功能的父类PraiseButton.
2. 开发子类Thumb实现大拇指方式点赞
3. 使用Babel编译ES6代码，并使用System.js可加载对应编译后的文件
4. 将编译后的文件挂载为jQuery的组件
5. 实现Karma完成对点赞+1组件的单元测试

```
  1. 先克隆项目 git clone
  2. npm install 安装依赖
  3. npm run dev 打开浏览器输入 http://localhost:3000


  单元测试(需要全局安装模块)
  npm install karma -g
  npm install karme-core -g
  npm install karma-jasmine --save
  npm install karma-chrome-launcher --save

  执行
  karma start

```
